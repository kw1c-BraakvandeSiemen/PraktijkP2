Eerste project site

Deze site was samen gemaakt met 2 andere medestudenten, de pagina die je op de foto ziet is door mij gemaakt. Het was best een grote en moeilijke opdracht want in de schets hadden we de homepagina best moeilijk gemaakt met een image slider, waar ik heel veel tijd aan kwijt was, maar ook veel van geleerd heb in dit project heb ik ook mijn aller eerste navigatie balk gemaakt en ook geleerd hoe elementen te positioneren zijn.

Tweede project site

Bij deze opdracht was het de bedoeling om de huisstijl te onderzoeken van veschillende websites en vervolgens zelf een nieuwe huisstijl samen te stellen voor de website van het Koning Willem 1 College, ik had de kleur rood vervangen voor de kleur oranje omdat deze in mijn optiek beter bij passende kleur emoties had, ook had ik het logo aangepast. Ik heb hier geleerd hoe ik een dropdown menu kan maken via HTML en CSS, verder heb ik geleerd hoe te werken met section elementen dat zijn de donkerblauwe en lichtblauwe vakken.

Complexe tabel

Dit is de eerste "moeilijke" tabel die ik heb moeten maken, hier was de bedoeling om de tabel te stylen zoals hij werd weergegeven op een voorbeeld, in het begin was het best lastig maar uiteindelijk was het makkelijk te doen ik heb hier mee geleerd hoe je een tabel aanmaakt en dus kan stylen.

Gamelijst via JavaScript

Dit ziet er misschien uit als een paar simpele regels tekst maar hier achter zit een hoop code via JavaScript die reageert op het aantal games wat de gebruiker wilt zien (1 tm 4) en op prijs volgorde (hoog of laag), daarnaast kan de gebruiker ook nog een gametitel invoeren en als die onder de games op de lijst valt toont de pagina op de 3e regel waar in de lijst de game staat, maar ook reageert de pagina als er een titel ingevoerd word die niet bekend is. dit was een hele leerzame opdracht omdat hier werd gewerkt met arrays, if else en schrijven naar HTML.
